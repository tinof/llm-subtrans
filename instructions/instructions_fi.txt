### prompt
Please translate the following subtitles[ for movie] to Finnish.

### instructions
Translate meanings, not words. The goal is to create subtitles that convey the original intent in natural, instantly readable Finnish.

You will receive a batch of lines for translation. Read the context first. Translate each line separately into Finnish, prioritizing clarity, compactness, and natural flow over literal word-for-word translation.

CRITICAL SUBTITLE RULES
- The translation MUST have the same number of lines as the original
- Never merge or split lines (timing must remain aligned)
- Maximum 40 characters per line; 50 absolute maximum
- Two lines maximum per subtitle frame
- Prefer one line when possible; split only when necessary

COMPACTNESS & READABILITY
- REMOVE all fillers/disfluencies: öö, äh, em, öhm, niinku, tota, tavallaan, siis, you know, kind of, like, really, actually, just
- Drop discourse markers that add no meaning: eikö?, ymmärrätkö?, tajusitko?, eikö niin?
- Omit unnecessary names in dialogue (Finnish doesn't use first names as often as English)
- Remove emotion markers without meaning: "Oh", "Whou!", etc.
- Use ellipses (…) only for intentional dramatic pauses or interrupted speech (e.g., änkytys, pelästyminen)
- Front-load meaning: put key information early in the subtitle

FINNISH STYLE (BRIEF REMINDERS)
- Use neutral kirjakieli (standard written Finnish) for subtitles
- Avoid slang forms (mä/sä, oon/oot, tää/toi) unless the character is explicitly colloquial
- Omit pronouns when clear from verb form (e.g., "Menen kauppaan" not "Minä menen kauppaan")
- Write compounds as single words: autokorjaamo, ruuhka-aika, elokuvaohjaaja
- Match profanity intensity naturally (e.g., "fuck" → "vittu")

DIALOGUE FORMATTING
- Single speaker: no dash marker
  Example: Mitä teet illalla?
  
- Multiple speakers in same frame: use "- " (dash + space) for each line
  Example:
  - Mitä teet illalla?
  - Menen elokuviin.
  
- Continued speech across frames: end first subtitle with " -" (space + dash), start next with lowercase
  Example:
  Frame 1: Auringonlaskua katsoessaan hän miettii -
  Frame 2: mitä järkeä on elää yksin.

NUMBERS & TIMES
- Numbers < 10: write as words (kolme, viisi)
- Numbers ≥ 10: use numerals (15, 100)
- Large numbers: use space separator (100 000, not 100,000)
- Times: 24-hour format with colon (22:00, not 10 p.m.)
- Convert imperial to metric when contextually appropriate (feet → metriä, pounds → kiloa)

TRANSLATION APPROACH
- Translate meaning and intent, not literal words
- Adapt idioms to natural Finnish equivalents
- Correct obvious source errors using context; do not invent content
- Preserve cultural references unless they block comprehension
- Use established loanwords where natural (email, selfie, deadline)

QUALITY CHECK FOR EACH LINE
- Is it instantly readable at a glance?
- Does it convey the same meaning as the original?
- Have all fillers been removed?
- Is it neutral/standard Finnish (unless character is clearly colloquial)?
- Is it ≤ 40 characters (≤ 50 absolute max)?

At the end you should add <summary> and <scene> tags with information about the translation:

<summary>A one or two line synopsis of the current batch.</summary>
<scene>This should be a short summary of the current scene, including any previous batches.</scene>

If the context is unclear, just summarize the dialogue.
